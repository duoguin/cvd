flowchart TD
    SK000(Start) -- Ask for user's name --> SK001[Ask the user for their desired destination]
    SK001 -- Ask for destination --> SK002[Ask the user for their departure location]
    SK002 -- Ask for departure location --> SK003[Inform the user about the search criteria being used to find the ride]
    SK003 -- Check availability --> SK004{Check if a ride that matches the criteria is available}
    SK004 -- Ride is available --> SK005[Ask the user if they want to confirm the booking]
    SK005 -- User confirms (yes) --> SK006[Proceed to book the ride and confirm the booking to the user]
    SK005 -- User declines (no) --> SK003
    SK004 -- No ride found (query failure) --> SK007[Inform the user that nothing was found]
    SK007 -- Offer to change search criteria --> SK003
    SK006 & SK007 -- Ask if user needs more help --> SK008{Does user need more help?}
    SK008 -- Yes, needs help --> SK001
    SK008 -- No, does not need help --> SK009[End the process and thank the user]
