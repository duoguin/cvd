flowchart TD
    SK000(Start) -- Ask for user details --> SK001[Ask the user for their name and the booking number of the ride they wish to change]
    SK001 -- Ask for change details --> SK002[Ask the user what changes they want to make to their booking]
    SK002 -- Attempt to make changes --> SK003[Attempt to make the requested changes to the booking by calling ride_change API]
    SK003 -- Changes successful --> SK004[Inform the user that their changes have been successful. Ask if there is anything else they need assistance with]
    SK003 -- Changes failed --> SK005[Inform the user that the attempt to change their booking has failed. Ask if there is anything else they need assistance with]
    SK004 & SK005 -- User needs further assistance --> SK001
    SK004 & SK005 -- No further assistance needed --> SK006[End the process and thank the user for using the service]
