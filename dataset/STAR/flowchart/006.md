flowchart TD
    HR000(Start) --> HR001[Ask the user for their name]
    HR001 --> HR002[Ask the user to choose a hotel]
    HR002 --> HR003[Ask the user for the check-in date]
    HR003 --> HR004[Ask the user for the check-out date]
    HR004 --> HR005[Ask the user for any specific requests they have for their stay]
    HR005 --> HR006[Check the availability of the hotel for the specified dates]
    HR006 -- Hotel is available --> HR007[Ask the user to confirm the booking]
    HR007 -- User confirms --> HR008[Attempt to book the hotel]
    HR008 -- Booking succeeded --> HR009[Notify the user that the reservation succeeded]
    HR008 -- Booking failed --> HR010[Notify the user that the reservation failed]
    HR007 -- User does not confirm --> HR002
    HR006 -- Hotel is unavailable --> HR011[Inform the user that it is unavailable]
    HR011 --> HR002
    HR009 & HR010 --> HR012[Ask the user if they need assistance with anything else]
