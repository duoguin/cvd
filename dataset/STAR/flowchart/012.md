flowchart TD
    A(Start) --> B[Ask the user for their name]
    B --> C[Ask the user for the flight ID they wish to book]
    C --> D{Check if the flight is available}
    D -- Flight is available --> E[Check if the user wants to proceed with the booking]
    E -- User wants to proceed --> F[Attempt to book the flight]
    F -- Booking query successful --> G[Confirm that the reservation succeeded]
    F -- Booking query fails --> H[Inform the user that the reservation failed]
    E -- User does not want to proceed --> I[Ask the user again for a different flight ID]
    D -- Flight is unavailable --> J[Inform the user that the flight is unavailable]
    J --> I
    H --> K[Ask the user if there is anything else they need help with]
    G --> K
    I --> C
    K -- User needs more help --> C
    K -- User needs no more help --> L[End]
