flowchart TD
SK000(Start) -- User wants to inquire about a city --> SK001[Call get_service_point_city_list to retrieve the list of service point cities]
SK001 -- List retrieved --> SK002[Check if the city provided by the user is in the supported list]
SK002 -- City is supported --> SK003[Call get_service_points to retrieve service point information for the city]
SK003 -- API call successful --> SK004[Return the queried service point information to the user. Ask if the user wants to inquire about another city]
SK003 -- API call unsuccessful --> SK005[Inform the user that no service point information is currently available for that city. Suggest querying nearby cities. Ask if the user wants to inquire about another city]
SK004 & SK005 -- User wants to inquire another city --> SK002
SK004 & SK005 -- User does not inquire another city --> SK007[The user is welcome to contact again for future needs]
SK002 -- City is not supported --> SK006[Prompt the user to choose the correct service point city. Provide self-service query links for PC and mobile. Ask if the user wants to inquire about another city]
SK006 -- User wants to inquire another city --> SK002
SK006 -- User does not inquire another city --> SK007[The user is welcome to contact again for future needs]
