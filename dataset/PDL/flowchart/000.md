SK000(Start) -- User specifies hospital, department, and time --> SK001[Verify the hospital name using check_hospital]
SK001 -- Hospital exists --> SK002[Verify the department name using check_department]
SK001 -- Hospital does not exist --> SK003[Notify the user that the hospital does not exist]
SK002 -- Department exists --> SK004[Query available appointment slots using query_appointment]
SK002 -- Department does not exist --> SK005[Notify the user that the department does not exist]
SK004 -- Slots available --> SK006[Attempt to make an appointment using register_hospital]
SK004 -- No slots available --> SK007[Query available slots at other hospitals using recommend_other_hospitals]
SK006 -- Registration successful --> SK008[Notify the user of the successful appointment]
SK006 -- Registration failed --> SK009[Notify the user of the failed appointment]
SK007 -- Slots available at other hospitals --> SK010[Ask the user if they are willing to make an appointment at another hospital]
SK007 -- No slots available at other hospitals --> SK011[Notify the user that there are no available slots]
SK010 -- User willing to make an appointment at another hospital --> SK012[Attempt to make an appointment at another hospital using register_other_hospital]
SK010 -- User not willing to make an appointment at another hospital --> SK013[Notify the user of the refusal to make an appointment]
SK012 -- Registration successful at another hospital --> SK014[Notify the user of the successful appointment at another hospital]
SK012 -- Registration failed at another hospital --> SK015[Notify the user of the failed appointment at another hospital]
